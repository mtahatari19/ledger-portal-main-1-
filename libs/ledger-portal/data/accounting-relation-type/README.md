# ledger-portal-data-accounting-relation-type

This library contains the data models and services for accounting-relation-type feature.

## Services

- `AccountingRelationTypeService`: Handles all HTTP requests for accounting-relation-type operations

## Models

- `AccountingRelationType`: Accounting relation type model
- `AccountingRelationTypeListResponse`: Response model for list endpoint
- `CreateAccountingRelationTypeRequest`: Request model for create endpoint
- `UpdateAccountingRelationTypeRequest`: Request model for update endpoint
